# Strings

  Strings são sequencias de caracteres terminadas por um caractere espericial: o nulo (```\0```). Elas não são um tipo de dado nativo da linguagem como em linguagens mais modernas (ex: Python ou Java). Em vez disso, strings em C são representadas como vetores de caracteres.
