# Operador Vírgula (,)

  Na linguagem C, o operador **vírgula (,)** pode ser utilizado de duas maneiras:

   - Como pontuação. Por exemplo, para separar argumentos de uma função:

       ```int minha_função(int a, float b);```

   - Para determinar uma lista de expressões que devem ser executadas sequencialmente:

       ```x = (y - 2, y + 3);```

     Nesse caso, as expressões são executadas da esquerda para a direita: o valor 2 é atribuído a y, o valor é somado a y, e o total (5) será atribuído à variável x. Pode-se encadear quantos operadores "," forem necessários.

     ### Na Linguagem C, o operador "," é um separador de comandos, enquanto o operador "," é um terminador de comandos.
