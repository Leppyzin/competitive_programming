# Tapetes

 O problema do tapetão deu trabalho para muita gente KKKKKKKK

 Achei um problema bem divertido e treina bastante a lógica


 ```
#include <stdio.h>

int main(){
    long long int total,aux,l,n,sa;
    
    scanf("%lld%lld",&l,&n);
    
    aux = n-1;
    
    sa = ((l-n+1));
    
    total = sa*sa+aux;
    
    printf("%lld\n",total);

    return 0;
}
```
