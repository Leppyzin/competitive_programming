# O que é um Algoritmo?

Um algoritmo é um conjunto de passos que devem ser executados de maneira lógica e bem estruturada, especificando passo a passo uma solução para um problema, sendo fundamental para o desenvolvimento de programas. Além disso, ele é sempre finito. Um algoritmo transforma informações de entrada (input) em um resultado (output) seguindo regras precisas. Na computação, eles são escritos em linguagens de programação e podem ser classificados em três tipos principais:

 - **Sequenciais:** Existem instruções em uma ordem direta, do início ao fim.
 - **Condicionais:** Tomam decisões baseadas em regras (Ex.: se a senha estiver correta, *então* liberar o acesso, *senão* negar).
 - **De Repetição:** Executam um bloco de instruções várias vezes até que uma condição seja alcançada.

### Exemplos no Dia a Dia

  Eles estão na base de toda a tecnologia moderna:

   - **Redes Sociais:** Definem as ordens de postagens e vídeos exibidos com base nos seus interesses.
   - **Aplicativos de Trânsito:** Calculam a rota mais rápida considerando a distância e o tráfego em tempo real.
   - **Mecanismos de Busca:** Filtram bilhões de páginas na internet para encontrar os resultados mais relevantes.
   - **Bancos:** Identificam automaticamente padrões suspeitos de compra para evitar fraudes.

## Problema inicial

 O seu primeiro programa em qualquer linguagem de programação normalmente é o "Hello World!". 
Neste primeiro problema tudo o que você precisa fazer é imprimir esta mensagem na tela("Hello World!").



### Algoritmo Descritivo

```
1) Imprima na tela a seguinte mensagem "Hello World!"
```

### Pseudo-código

```
Imprima na tela "Hello World!"
```
